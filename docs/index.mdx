# Bossa

The memory layer for your agents. Persistent, workspace-scoped filesystem — `ls`, `read`, `write`, `grep`, `glob` backed by Postgres.

<Callout icon="leaf" color="#16A34A">
**Base URL:** `https://filesystem-fawn.vercel.app` · No infrastructure to run. Sign up, get an API key, start building.
</Callout>

## Why Bossa?

<Columns cols={2}>
  <Card title="Memory as filesystem" icon="folder" href="/GETTING_STARTED">
    Agents understand files. Use familiar operations — no new abstractions.
  </Card>
  <Card title="Dynamic context discovery" icon="search" href="/AGENT_INTEGRATION">
    Search at runtime. Find what matters without packing static prompts.
  </Card>
  <Card title="CLI + MCP + REST" icon="terminal" href="/MCP">
    Subprocess agents, Claude/Cursor/LangChain, or scripts.
  </Card>
  <Card title="Workspace isolation" icon="lock" href="/GETTING_STARTED">
    Each agent gets its own space, scoped by API key.
  </Card>
</Columns>
